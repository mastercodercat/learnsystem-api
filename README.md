# LearnSystem API

## Application Dependencies

- Express.js

- Prisma

- TypeScript

## How to start

- First, copy `.env.example` file and rename it to `.env`

- Then, run `npm install`

- Then, run `npm run dev` or `npm start`

## Log module

- `morgan`

- `winston`

## Authenticate/Authorize

- `passport`

## Validation

- `express-validator` for request validation
